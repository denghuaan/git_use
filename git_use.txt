1/在你需要放项目的目录中，点击右键 git bash here

2/git clone 仓库地址（右上角的绿色按钮，复制里面地址）

3/cd进入新创建的目录

4/把你修改的内容提交 
	4.1/git add 指定文件目录  
	4.2/git commit -m "提交的是什么操作"
	4.3/git push -u origin master

5/更新git pull
